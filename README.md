# dotfiles

This is the configuration of my Arch linux based installation with Sway or Gnome

## Common Packages

- Terminal: alacritty
- Editor: nvim with lazyvim
- Icons: Font Awesome
- Menus: Rofi
- Browsers: brave, chromium
- Filemanager: ranger, Thunar

## Sway
	- swaylock
	- rofi
	- rofi-power-menu
	- swaybg
	- wlsunset
	- wl-clipboard
